## 1. 목표

산업별 "예측을 잘하는" 애널리스트 Top 3를 유지·관리하고, 이들의 리포트 발행을 즉시 감지하여 (a) 원문이 공개되면 내용 요약을, (b) 비공개면 출시 사실과 핵심 메타데이터를 대시보드와 알림으로 제공한다.

## 2. 애널리스트 선정 체계

### 2.1 평가 소스 비교 (2026-06 검증)

| 소스 | 성격 | 산업별 제공 | 자동화 가능성 | 용도 |
|---|---|---|---|---|
| 한경컨센서스 애널리스트 스코어 | **정확도 기반** (목표가-실제주가 괴리율 → 적중률, 평균 변동률, 리포트 수 가중. 1/3/6/12개월) | 직접 제공 안 함 (애널리스트 상세에서 커버 종목 확인) | 낮음 — 전체 목록 로그인 필요, 무단 배포 금지 약관 | 분기 1회 수동 참조 |
| 매경 베스트 애널리스트 | 설문 기반, 연 1회, 35개 부문 | O | 기사로 공개 → 수동 입력 | 시드 |
| 한경비즈니스 베스트 애널리스트 | 설문 기반, 반기별, 섹터별 | O | 기사로 공개 → 수동 입력 | 시드 |
| 에프앤가이드 베스트 애널리스트 | 설문 기반, 13개 산업 부문 + 7개 전략 부문 | O | 공개 페이지 → 수동 입력 | 시드 |
| **자체 적중률 엔진** | **정확도 기반**, 우리 이벤트 DB로 계산 | O (종목→산업 매핑 보유) | **완전 자동** | 본 시스템의 최종 기준 |

핵심 판단: 어워드는 기관투자자 인기/신뢰 투표이지 예측 정확도가 아니다. 정확도 기반 평가는 한경 스코어와 자체 엔진뿐이며, 한경 방법론이 공개되어 있으므로 동일 사상을 자체 구현한다.

### 2.2 자체 적중률 산출 방법론 (한경 방식 차용 + 보완)

이벤트 DB의 각 리포트 r = (발행일, 애널리스트, 종목, 목표주가 TP, 발행일 종가 P0)에 대해:

```
적중률(r, h) = 100% - |TP - P(h)| / TP        # h = 6개월, 12개월 시점 주가
적중 여부    = 적중률 ≥ θ (초기값 θ=85%, 백테스트로 보정)
방향 점수    = sign(TP - P0) == sign(P(h) - P0) ? 1 : 0   # 보완: 방향 적중 별도 측정
애널리스트 정확도(산업 i) = 산업 i 종목 리포트 중 적중 비중
가중 스코어  = z(정확도) × 0.5 + z(방향점수) × 0.3 + z(리포트 수) × 0.2
```

보완 사항: (1) 목표가는 통상 12개월 전망이므로 6개월/12개월 이중 측정, (2) 괴리율만 보면 보수적 목표가가 유리해지는 왜곡이 있어 방향 적중을 별도 반영, (3) 산업별 리포트 5건 미만 애널리스트는 랭킹 제외(소표본 왜곡 방지).

### 2.3 산업별 Top 3 운영 (2단계)

**Phase A — 부트스트랩 (서비스 시작 ~ 6개월)**
- 시드: 어워드 3종의 산업별 수상자 + 한경 스코어 수동 참조로 산업당 3명 등록
- 검증된 시드 예시(2026-06 확인): 자동차 — 장문수(현대차증권, 매경 2년 연속 + 한경비즈니스 1위), 인터넷·소프트웨어 — 임희석(미래에셋증권, 2025 상반기 한경비즈니스)
- 어워드 부문 ↔ 우리 Level 2 매핑 테이블 필요 (예: "조선/중공업/기계" 부문 → 조선·기계·로봇 2개 산업에 공동 시드)
- 시드가 없는 산업(부문 미존재): 해당 산업 리포트 발행 빈도 상위 3명으로 대체

**Phase B — 자동 선정 (6개월 후~)**
- 자체 적중률 엔진의 산업별 가중 스코어 상위 3명으로 분기마다 자동 갱신
- 교체 시 대시보드에 변경 이력 표시 (누가 빠지고 누가 들어왔는지, 스코어 근거)
- 어워드 수상은 보조 배지로만 표시

## 3. 리포트 수집: 3-Tier 폴백 구조

```
[감지] 네이버 리서치 목록 폴링 (장 마감 후 일 1회 + 오전 1회)
   → 신규 행: (날짜, 증권사, 제목, 종목/산업, 첨부 PDF 유무, 상세 링크)
   → 상세 페이지: 애널리스트명, 목표주가, 투자의견
   ↓
[Tier 1] PDF 공개분
   → 다운로드 → 텍스트 추출 → LLM 요약 (야마 1줄 + 근거 3줄 + 목표가 변경 맥락)
   → 원문은 로컬 보관(개인 분석용), 요약 카드 생성
[Tier 2] PDF 비공개 (증권사 고객 전용 등)
   → 메타데이터 카드만 생성: 제목, 증권사/애널리스트, 목표가 변경(직전 대비 화살표), 원문 링크
   → 카드에 "원문 미공개 — 출시 알림" 배지
[공통] 워치리스트 매칭
   → 발행 애널리스트가 산업별 Top 3에 포함되면: 대시보드 피드 상단 고정 + 텔레그램 즉시 푸시
   → Top 3 외 리포트도 이벤트 DB에는 전량 적재 (시그널 계산용)
```

가능성 정리: **출시 감지와 알림은 전체 리포트에 대해 보장**(목록 메타데이터는 항상 공개). 원문 요약은 공개분에 한정되며, 비공개 비중은 운영 데이터로 측정 후 필요 시 한경컨센서스를 보조 소스로 이중화.

### 3.1 동일인 식별 문제

애널리스트 이름은 동명이인·표기 변형(공동 저자 병기 등)이 있으므로:
- `analysts` 마스터에 (이름, 증권사) 복합키로 등록, 이직 시 새 레코드 + `predecessor_id` 연결
- 공동 저자 리포트는 제1저자 기준 귀속, 전체 저자는 별도 배열로 보관

## 4. 대시보드 반영

| 위치 | 내용 |
|---|---|
| 메인 피드 "애널리스트 워치" | Top 3 애널리스트의 신규 리포트 카드 스트림. Tier 1은 요약 펼침, Tier 2는 메타데이터 + 링크 |
| 산업 상세 | 해당 산업 Top 3 프로필(스코어, 최근 적중 사례, 커버 종목) + 리포트 타임라인 |
| 애널리스트 상세 | 적중률 추이 차트, 목표가 제시 이력 vs 실제 주가 오버레이 |
| 알림 | Top 3 신규 리포트(즉시), 목표가 10%+ 상향(즉시), Top 3 교체(분기) |

```
               direction_rate, n_reports, weighted_score)
```

## 6. 법적·운영 유의사항

- 리포트 PDF는 개인 분석 용도로만 로컬 보관, 요약 카드도 본인 대시보드 한정 (재배포 금지)
- 한경컨센서스 애널리스트 스코어는 약관상 자동 수집 부적합 → 수동 참조로 한정
- 어워드 정보는 공개 보도 기반으로 수동 입력 (반기 1회 운영 루틴)

## 7. 로드맵 반영 (기획서 갱신 필요 사항)

- Phase 1에 `analysts` 마스터 + 시드 입력 추가 (공수 작음)
- Phase 3의 알림에 "애널리스트 워치" 트리거 추가
- Phase 4의 "주요 애널리스트 선정 자동화"는 자체 적중률 엔진으로 구체화 — 어워드 부트스트랩 덕분에 서비스 첫날부터 워치리스트 가동 가능

## 8. 미결 사항

- [ ] 적중 임계값 θ와 가중치(0.5/0.3/0.2)의 백테스트 보정 계획
- [ ] 비공개 PDF 비중 측정 후 한경컨센서스 이중화 여부 결정
- [ ] 산업 리포트(종목 미특정)의 애널리스트 평가 반영 방법 — 목표가가 없어 적중률 계산 불가, 발행 빈도만 반영할지
- [ ] 어워드 부문 ↔ Level 2 매핑 테이블 초안 작성
- [ ] **증권사 리서치센터 직접 링크(broker_url) 확보율 검증** — 한경컨센서스/네이버가 증권사 도메인 원문 링크를 얼마나 제공하는지 Phase 1에서 실데이터로 확인. 확보 안 되는 증권사는 "증권사명 + 검색 안내" 정도로 폴백
